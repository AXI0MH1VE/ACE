# Finance Axiom Set (GAAP2025)

- All journal entries must balance (total debits == total credits).
- Revenue recognition must reference contract_id and recognition_schedule.
- Expense capitalization requires asset_class and amortization_period.
- Cashflow statements must reconcile net income with operating cash via adjustments.
- Fraud guard: entries above 5% materiality require dual signatures + timestamp.
- ZK policy: C=0 signature must include axiom_hash of this document and proof_cert from Lean/EZKL.
