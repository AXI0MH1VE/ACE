Security Policy

Reporting
- Report vulnerabilities to [your-email@example.com] with [SECURITY] in subject.
- Include repro steps, impact, and proof-of-concept.

Scope
- In-scope: core chatbot logic, safety, broker, ledger.
- Out-of-scope: third-party deps (report upstream), user misconfiguration.

Updates
- Dependencies pinned and audited via uv lock; check for updates with uv sync.
